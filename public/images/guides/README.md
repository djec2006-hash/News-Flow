# Images des Guides

Ce dossier contient les captures d'écran utilisées dans les guides de la page d'aide.

## Images requises

Placez les images suivantes dans ce dossier :

1. **configurer-sources.jpg** - Capture d'écran de la page de configuration des sources
2. **creer-premier-flow.jpg** - Capture d'écran de l'interface de création de Flow
3. **exporter-partager.jpg** - Capture d'écran montrant un Flow avec les options d'export
4. **gerer-projets.jpg** - Capture d'écran de la liste des projets
5. **configurer-emails.jpg** - Capture d'écran des paramètres de notification email
6. **comprendre-plans.jpg** - Capture d'écran de la page de tarification

## Format recommandé

- Format : JPG ou PNG
- Dimensions : 800x600px minimum
- Poids : < 500KB par image
- Style : Capture d'écran claire avec annotations si nécessaire

## Note

Si une image n'est pas trouvée, un placeholder avec l'icône du guide sera affiché automatiquement.
